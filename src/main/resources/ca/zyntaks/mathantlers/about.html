Brief Syntax Guide

Invisible grouping: { a + b } / c will put a + b together above c.
Parentheses: (a + b), {{a + b}}, or [a + b].
Exponents: a^b
Multiply (dot): a*b
Divide (fraction): a/b
Add, Subtract: a+b, a-b
Floor, ceil: floor(a), ceil(a)
Subscript: a_i
Set notation: {{ x el R | x < 5 }}
Summation: sum(i = 0, 6)
Align Lines by Equals Sign:
Use == instead of = to align.
Also works with <=, >=, <, >, != by adding a = before them.
Function with multiple cases:
f(x) == cases {
    x + f(x - 1), x > 0;
    0, x< 1;
}
